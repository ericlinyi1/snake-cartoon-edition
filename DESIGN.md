# Snake Cartoon Edition: Architecture & Tech Stack

## Tech Stack
- **Frontend**: HTML5 Canvas, Vanilla JavaScript (or minimal framework like Vite)
- **Styling**: Tailwind CSS (for layout/cartoon UI)
- **Container**: Docker + Nginx
- **CI/CD**: GitHub Actions
- **Deployment**: Hostinger VPS (Docker)

## Game Engine Design
- **Grid-based movement**
- **Collision Detection**: Wall, Self, Food
- **Cartoon Aesthetics**: 
  - Rounded snake segments
  - Vibrant colors
  - Particle effects for "eating"
  - Shake screen on death

## Mobile Support
- Touch controls (D-pad overlay)
- Responsive canvas sizing
